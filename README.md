# deal
Monopoly deal in JS/HTML/CSS

Chat: Socket.IO

# Installation
Run `npm install`

To run use `node --use-strict index`

visit localhost:3000
Click "New Game"
Click "Draw Cards"


#Features so far.
* Create a full deck
* Shuffle Deck
* Chat with other players
* Pick up cards from deck to hand
* Add money cards to bank
* Add cards to inPlay 
* Play a card into center
